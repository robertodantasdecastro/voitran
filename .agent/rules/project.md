# Project Rule

## Stack
- `SwiftUI` no `macOS`
- `Go` no backend/control-plane
- runtime externo em `/Volumes/SSDExterno/Voitran_runtime`

## Regras do projeto
- priorizar latencia percebida pelo usuario;
- manter modelos e caches fora do repo;
- tratar consentimento de voz como contrato, nao como opcional.
