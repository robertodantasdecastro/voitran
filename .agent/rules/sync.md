# Sync Rule

- toda mudanca em `apps/`, `backend/`, `packages/`, `scripts/`, `.agent/`, `.antigravity/` ou `docs/memory/` exige revisao da memoria viva;
- toda divergencia entre `Mac`, `Backend` e `AntigravityIDE` deve entrar no `AGENT_SYNC_BOARD`;
- todo checkpoint relevante deve atualizar `CHANGELOG` e `project_evolution`.
