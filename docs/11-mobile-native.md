# Mobile Native

## Objetivo
Planejar a expansao para `iOS` e `Android`.

## Direcao
- `iOS` reutiliza `packages/realtime-core-swift`;
- `Android` usa base `Kotlin + NDK`, com portabilidade via `whisper.cpp`;
- degradacao por `capability tier` desde o inicio.

## Saidas esperadas
- alpha iOS;
- base nativa Android;
- negociacao de capacidade comum;
- mesmas politicas de consentimento e identidade vocal.
